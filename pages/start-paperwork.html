Start Paperwork

All paperwork must be submitted digitally through TiM (hellotim.com). This is where you can send, manage, and submit all crew start paperwork for your production.

If you don’t already know how to use TiM, we can provide a training session to get you onboarded. Please contact your Production Coordinator to set up a training session with them.
