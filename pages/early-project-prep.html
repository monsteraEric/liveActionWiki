Be prepared to discuss and/or share the following preliminary documentation and any special insurance requirements to gain authorization for the next steps in Production Prep.

Contact information for all production personnel

Crew and cast list MUST be filled out completely

Calendar

Travel itineraries

Special Insurance Requests

Server Organization

Any documentation or digital assets created in prep, pre-production, or on shoot must be filed properly in the google folder provided by Lord Danger.

Accounting

All POs, receipts, invoices, W9’s should go here.

See PO section for organization

Find Lord Dangers W9 in this folder

Budgets

Running budgets should go here.

Cast/ Crew Lists

All cast and crew lists should go here.

They must be updated daily.

Client

Keep client, logos, fonts, and anything received from client in this folder for pre-pro book or reference.

Contracts & Releases

Template agreements should go here

Executed Talent agreements and Executed Crew deal memos should go into this folder and be marked as executed in a separate folder once completed

Any location agreements, releases, contracts should go into this folder and be marked as executed in a separate folder once completed

Insurance

All generated COI’s should go here

Permits

All permits and supporting documents should go here

Production Department Shared Folders

All internal paperwork should go here

(For internal paperwork and photos for art, wardrobe, ad’s, call sheets etc.)

Scripts/ Creative

All scripts, storyboards, and, decks should be kept in this folder with clearly marked versions

The newest and/or final creative should go in a “Final Creative” folder for easy access

Wrap

Early Working Actuals

The Line Producer Must Submit a Working Actual twice during Shoot Prep

The day of, or morning after the Tech Scout

The day before the production’s first shoot day

If there are foreseeable or anticipated budget issues prior to submitting drafts of the working actual, please loop in the Production Manager AND the AP as soon as possible. The sooner any potential situation can be brought to our attention the sooner we can assist.

Use the following format as the subject for delivery:

JOB NAME / JOB # / Subject / Current Date

Pre-Production Book

Lord Danger requires Pre-pro books to be digital unless specifically requested by agency

PrePro book template can be found in the project’s Drive under “Client”, labeled “PrePro Binder”

Send the PrePro book to the AP/EP the day prior to the PPM for approval before sending it to Agency/Client

Should the PPB undergo changes after the PPM, please re-send the final version of the PPM
