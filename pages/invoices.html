Invoices

When projects are underway or wrapping, invoices will need to be submitted to accounting for payment. Note: We like to maintain net 30 terms for all invoices from companies/vendors and Net 15 terms for all freelance artists so please try and let them know of that policy before approving an estimate/quote. If the vendor does not agree, then we can discuss alternative terms for payment.

Due to the pandemic and constant delays from the mail systems, we want to ensure we set up everyone with ACH Direct Deposit. You will find the ACH form included here: 04_shoot⁩ ▸ ⁨07_templates_resources⁩ ▸ ⁨_resource_templates⁩ ▸ ⁨additional_templates⁩ > ACH Deposit Authorization via ACH

We also require a current W9 form from each vendor invoicing. Please be sure to have the vendor include a current W9 form when they send their invoice and ACH form back.

The process for sending invoices to accounting is simple and should speed up the process as well as ensure no invoices slip through the cracks.

HOW TO ORGANIZE INVOICES

All Invoices are treated as POs and will need to be organized via a PO Log and subsequent folders inside 04_shoot > 01_purchase_orders

Here, you’ll see a folder for the log. You should have already been briefed on how to use the PO Log but if not, please contact caleb.bell@lorddanger.com and we’ll set up a quick tutorial.

The PO backup documents are to be placed inside a PO folder titled: 001_ExampleCo, 002_Jane_Doe, etc.

These docs include

Invoice

W9

PO Top Sheet (Generated by the PO Log)

ACH Form (Try your best to get as many as you can).

Any receipt scans for a reimbursement.

Etc.

Ensure we keep the log updates as this is what Accounting uses to track and monitor invoices. Ensure the payment terms are correct as well (quickpay, net 30, net 15, etc). As long as each PO is labeled correctly with the correct due dates, accounting will ensure these get handled.

Accounting will then reference the associated PO folder and make sure they gather all documents needed for the payment.

Once the invoice has been scheduled for payment, accounting will check it off on the log to let everyone know it’s been dealt with.

Any POs left unchecked should be treated as unpaid and will require a follow-up to accounting. Never assume it’s been handled if it's unchecked.

PING ACCOUNTING WHEN INVOICES START COMING IN

When invoices start coming in and accounting needs to start tracking, we need to let them know.

Send an email to billing@monstera.group

Subject: “000_CLIENT_NAME // POs to Pay”

Body: In the body, include a quick write up letting them know invoices are starting to come in and we’d like their eyes on it to begin to track these. Include a direct link to the PO log as well a link to the purchase order
